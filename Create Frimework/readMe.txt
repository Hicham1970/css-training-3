Les framework les plus connus dans le Css sont Bootstrap et Tailwind .
Bootstrap est un framework CSS et HTML qui permet de créer des sites web responsives et modernes.
Il est très populaire et utilisé par de nombreux développeurs web.
Tailwind est un framework CSS qui permet de créer des sites web personnalisés et flexibles.
Il est également très populaire et utilisé par de nombreux développeurs web.
Les avantages de Bootstrap :
* Facile à utiliser et à apprendre.
* Possibilité de créer des sites web responsive et modernes.
* Grande communauté et nombreux ressources disponibles.
* Compatible avec la plupart des navigateurs.
Les avantages de Tailwind :
* Permet de créer des sites web personnalisés et flexibles.
* Possibilité de créer des composants réutilisables.
* Grande communauté et nombreux ressources disponibles.
* Compatible avec la plupart des navigateurs.
Les inconvénients de Bootstrap :
* Peut être lourd et ralentir le chargement du site.
* Peut être difficile à personnaliser.
* Peut avoir des problèmes de compatibilité avec certains navigateurs.
Les inconvénients de Tailwind :
* Peut être difficile à apprendre et à utiliser pour les débutants.
* Peut nécessiter une grande quantité de code pour créer des éléments de base.
* Peut être difficile à personnaliser.


Et pourquoi ne pas créer notre propre framework?
Créer son propre framework peut être une excellente idée si vous avez des besoins spécifiques, comme par exemple une classe ou les éléments sont centrées dans le document 
.center {
    position:absolute; 
    left:50%;
    top:50%;
    transform: translate(-50%, -50%);
}
Cette classe sera ajouter a tous les éléments du documents ui nécessitent d'être centrés
